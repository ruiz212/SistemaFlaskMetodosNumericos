El backend en Python interpreta, deriva y evalúa automáticamente. Se acabó programar derivadas a mano — el sistema lo resuelve algebraicamente en tiempo real.

import sympy as sp

# Derivación automática
expr = sp.sympify("x**3 - 3.5*x**2 + x")
J = sp.diff(expr, x)

# Evaluación numérica
val = float(J.subs(x, 2.0))
